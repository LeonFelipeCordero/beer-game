# Beer Game

To understand the beer game see [here](RULES.md)
## Java
### Build
```bash
./gradlew clean build -x test
```

### Test
```bash
./gradlew test
```

### Run
```bash
./gradlew bootRun
```

## Web App
### Code generation (Graphql)
Start backend application
```bash
npm run codegen
```

### Run
```bash
npm run dev
```
